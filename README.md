# Global Dynamics in Military Expenditure

## Project Overview
This project analyzes global military expenditure trends across countries and regions over time to identify key spending patterns and geopolitical insights.

## Tools & Technologies
- Python (Pandas, NumPy, Matplotlib, Seaborn)
- Google Colab
- Exploratory Data Analysis (EDA)
- Data Visualization

## Project Structure
- notebooks/ → Google Colab notebook with full analysis
- reports/ → PDF summary of findings
- presentation/ → PowerPoint presentation

## Dataset
Source: SIPRI / Kaggle  
Dataset not uploaded due to size or licensing restrictions.

## Key Outcomes
- Identified top military-spending countries
- Analyzed regional and temporal trends
- Visualized global defense spending patterns
